## Getting it running

1.  **Depencencies**

    ```sh
    npm install
    ```

2.  **Start developing**


    ```sh
    npm start
    ```

## Editor support for VS Code

    Install BuckleScript & Reason globally:

    ```sh
    npm install -g bs-platform
    ```

    Install the reason-vscode@1.7.4 and prettier vs code extensions.
